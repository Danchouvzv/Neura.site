📋 ШАГИ ДЛЯ ДЕПЛОЯ:

1. ЗАЙДИТЕ НА VERCEL.COM
   → Войдите через GitHub/GitLab/Bitbucket

2. ADD NEW PROJECT
   → Импортируйте ваш репозиторий

3. НАСТРОЙКИ (автоматически определятся):
   ✅ Framework: Vite
   ✅ Build Command: npm run build
   ✅ Output Directory: dist

4. ДОБАВЬТЕ ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ:
   Settings → Environment Variables → Add:
   
   VITE_SUPABASE_URL = ваш_supabase_url
   VITE_SUPABASE_ANON_KEY = ваш_supabase_anon_key
   GEMINI_API_KEY = ваш_gemini_key (опционально)

5. DEPLOY!
   → Нажмите кнопку "Deploy"

6. ПОСЛЕ ДЕПЛОЯ:
   → Скопируйте URL (например: https://your-project.vercel.app)
   → Добавьте его в Supabase CORS:
     Supabase Dashboard → Settings → API → 
     Additional Allowed Origins → Добавьте URL

🌐 ДОБАВЛЕНИЕ СОБСТВЕННОГО ДОМЕНА:

1. Settings → Domains → Add ваш домен
2. Скопируйте DNS записи от Vercel
3. Добавьте их в панели управления регистратора
4. Подождите 1-2 часа
5. SSL выдастся автоматически
